## Scope
- Audited file: `index.html`
- Audit tools requested: Chrome Lighthouse + WAVE

## Issues Found
1. Image alt text was missing.
2. Heading flow needed clearer hierarchy.
3. Some link text was not descriptive enough.
4. Page metadata needed better language/description support.

## Fixes Applied
1. Added meaningful `alt` text to images.
2. Kept a logical heading order (`h1` then `h2` sections).
3. Used descriptive link labels (for example, `project showcase` and `MDN Web Docs`).
4. Confirmed `lang="en"` and added a page `meta description`.
